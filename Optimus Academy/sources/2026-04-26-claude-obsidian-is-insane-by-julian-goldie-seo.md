## Summary

Julian Goldie pitches Claude + Obsidian as an "AI second brain" workflow — turning a vault of plain-text notes into an active system that reads, queries, and writes for you. Two integration paths are covered: lightweight Obsidian plugins (Claudian, Sier) for in-note chat, and Claude Code pointed at the vault folder for bulk operations and agentic automation. The video closes with three concrete business use cases (content creation, client work, learning) plus an "advanced moves" section on vault watching, role-based knowledge agents, and content generation from research.

## Detailed Notes

### Chapter 1 — Intro: Your AI Second Brain (0:00)

The opening promise: turn messy notes into an "AI brain" that writes, codes, and automates a whole business. No coding, no tech skills, free to start. Pitch is Claude + Obsidian as the combination that's "blowing up right now." Goldie frames it as a step-by-step setup show.

### Chapter 2 — What is Obsidian & Claude? (0:38)

**Obsidian as a different kind of notes app:**
- Notes live on your computer in plain text files (you own them forever, no cloud lock-in)
- Notes link to each other like a web of ideas
- Common framing: a "second brain"

**Claude as the AI partner:**
- Anthropic's model
- One of the best AI models out right now (Goldie's framing)

**What changes when you put Claude inside Obsidian:**
- Notes become "alive"
- Claude can read them, understand them, write new notes from them, answer questions about them
- With Claude Code, automation enters: scripts, systems, "real work"

**Concrete example use cases pitched in this section:**
- "Pull all my notes on my best clients. Find what they have in common. Write a plan to get more clients like them." — done in seconds.
- "Look at all my content ideas. Pick the best 10. Write YouTube scripts for each one." — 10 scripts from notes that were sitting idle.

### Chapter 3 — How to Set Up Your Vault (1:54)

**Step 1:** Download Obsidian. Free. obsidian.md. ~2 minutes to install.

**Step 2:** Create a new vault. A vault is just a folder where notes live. Name it anything (e.g. "business brain," "my AI vault").

**Step 3:** Add Claude. Two paths covered:

**Path A — Plugins (lightweight, in-note chat):**
- Obsidian has a plugin store
- Plugins to look at: **Claudian** and **Sier** (both let you chat with Claude inside notes)
- Setup: install plugin → add Claude API key → done
- Usage: highlight any note, ask Claude a question, get answer right inside the vault

**Path B — Claude Code (more powerful, terminal-based):**
- Covered in detail in the next chapter

### Chapter 4 — Power Up with Claude Code (2:33)

Claude Code is positioned as the real magic. Tool from Anthropic, runs in terminal. Can read files, write files, run commands, "do real work."

**Setup:** Point Claude Code at your Obsidian vault folder. Now it has full read/write access to all notes.

**Capability examples cited:**
- Reorganize 500 notes — Claude Code does it
- Find every note about a topic and produce a summary
- Build a whole content calendar from raw idea notes

The pivot point: plugins give you in-note Q&A; Claude Code gives you bulk operations and creation across the entire vault.

### Chapter 5 — 3 Real-World Use Cases (3:23)

**Use case 1 — Content creation:**
Dump every content idea (notes, docs, ideas in your head) into Obsidian. Be sloppy — "essie is fine" (his casual phrasing for "messy is fine"). Then ask Claude:
> "Look at all my content ideas in this folder. Group them by topic. Pick the top 20. Write hooks for each one."

Result: chaos → content plan in 5 minutes.

**Use case 2 — Client work:**
The pain: meeting notes, emails, project details scattered everywhere. The setup: one folder per client in Obsidian, dump everything into the right folder. Then ask:
> "Put together everything on this client. Make a one-page summary. Tell me what's next."

Result: walk into every client meeting prepared, look sharp, clients notice.

**Use case 3 — Learning (Goldie says this is the game-changer for him personally):**
Every time he learns something new — YouTube videos, books, podcasts, courses — notes go into Obsidian. Then ask Claude:
> "Take my notes on AI automation. Quiz me on the key ideas. Find gaps in my learning."

Claude becomes a tutor that knows what you know and what you don't. Faster learning, better retention, actual application.

**Claude Code workflow walkthrough:**
- Open terminal
- Install Claude Code (one command)
- Point at the Obsidian vault folder
- Issue commands like:
  > "Read every note in this folder. Find the ones about marketing. Make a new note called 'master marketing plan.' Put all the best ideas in there. Organize them by traffic, leads, and sales."

A full day of manual work → one minute via Claude Code.

### Chapter 6 — Advanced Workflow Automation (5:05)

Three "advanced moves" beyond basic vault Q&A:

**Advanced move 1 — Connect vault to your whole workflow (vault watcher):**
Tell Claude Code to *watch* your vault. Every time a new note is added, Claude does something with it automatically.
- Drop a meeting note in a folder → Claude reads it, pulls out action items, adds them to your task list
- "You did nothing. The system did it all."

**Advanced move 2 — Build a knowledge agent (role-based assistant):**
Set Claude up as a mini team member:
- Give it a role
- Give it rules
- Give it access to your vault

Then it answers questions like a team member would:
- "What did we decide about pricing last month?" → reads your notes and tells you
- "What's our content plan for this quarter?" → pulls everything together

**Advanced move 3 — Generate content from research (Goldie says he uses this every day):**
A vault folder full of research articles, YouTube transcripts, ideas. Point Claude Code at it:
> "Write me five YouTube scripts based on this research. Use my style. Use my hooks. Make them punchy."

Output: five full scripts in his voice, based on real research. "What used to take me all week now takes 10 minutes."

### Honest tradeoffs section (6:07)

Goldie is upfront about downsides:
- You need a Claude API key OR a Claude subscription (cost)
- Plugins can be buggy
- Claude Code has a "huge" learning curve — you have to get used to the terminal
- If your vault is huge, some tasks take longer

His take: the wins beat the downsides easily. "I'd rather spend a few bucks a month and save hundreds of hours."

### The closing thesis (6:23)

AI is moving fast. Winners are people who build systems now, not next year, not next month. Obsidian + Claude is positioned as one of the best systems you can build:
- Simple, cheap, powerful, grows with you
- Every note added makes it smarter
- Every month of use makes it more valuable

Contrast with the default workflow most people use: copy-pasting into ChatGPT.
- ChatGPT is fine but slow
- It forgets everything
- It doesn't know your business

Claude + Obsidian fixes all of that — the AI knows you, your business, your clients, your ideas.

### Chapter 7 — How to Start Today (7:03)

**Recommended starting sequence:**
1. Download Obsidian today
2. Start a vault — don't worry about being organized, just get notes in
3. Pick one plugin (Claudian is named as a good start)
4. Add your Claude API key, play with it
5. Once comfortable, try Claude Code — point it at your vault, ask it to do something big
6. Start small, build up
7. "In a few weeks, you'll have an AI brain that runs your business"

Closing CTAs (Goldie's products): AI Profit Boardroom (paid SOPs + use cases), AI Success Lab (free community of 68k+ members).

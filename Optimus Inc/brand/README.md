# Optimus Brand Identity

The brand constitution for Optimus Business Solutions itself. Drives the marketing site, the AI agent voices, the social content tone, and every prospect-facing surface.

This hub is mostly a stub right now. It gets filled when Optimus's own marketing site is built — that build forces the brand to be defined for real, the same way every client build forces a `design-system.md` to exist before code starts.

## Logo

Current asset: `Revamp logo light.png` lives at the vault root. Should move into this folder once the brand is properly defined. Out of scope for this pass.

Open: dark variant, monochrome variant, favicon, social avatar crops, PNG vs SVG sourcing.

TBD. Should be derived from a future Optimus `design-system.md`, built using the same `[[../../design-synthesizer]]` agent that produces client design systems.

Anchor points (working assumption, subject to the design-synthesizer pass):

- Premium but not corporate
- Specific over generic
